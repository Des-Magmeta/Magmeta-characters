# Magmeta-characters
Magmeta 开源字体字集初步整理一览表

## 简介 
  
 由于本表仅凭个人之力，以经验判断而成，若有未逮，在所难免。如各位有异议，或看到疏漏、讹谬之处，非常欢迎向在下提出，不吝赐正，以匡不逮。 
  
 > ㊟　目前的 Markdown 格式字表仅列出汉字，读音、释义、对应 Unicode 编码可通过[「字统网」](https://zi.tools/)或[「叶典」](http://yedict.com/)查询。
